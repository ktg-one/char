#   {emote-name}-mouth-closed.png   (resting face)
#   {emote-name}-mouth-open.png     (for lip-sync animation while speaking)

#   - Use the in-app Emotes modal (+ Add Custom Emote) to upload them, OR

## TIPS FOR CREATING MORE
- Always start with the exact BASE CHARACTER PROMPT for visual consistency.
- Add "square 1:1", "transparent background", "centered on face and shoulders".
- For lip-sync pairs: generate closed first, then edit the image (not text prompt) to open the mouth while keeping everything else identical.
- Recommended size: ~600x600 px (matches the current UI assets).
- After generating:
  1. Rename files to {name}-mouth-closed.png and {name}-mouth-open.png
  2. Copy into companion-python/static/images/custom/
  3. Either use the Emotes UI to add them, or append to data/emotes.json like this:

{
  "name": "playful",
  "keywords": ["playful", "tease", "mischievous", "wink"],
  "color": "#FF69B4",
  "images": {
    "closed": "/static/images/custom/playful-mouth-closed.png",
    "open": "/static/images/custom/playful-mouth-open.png"
  }
}

- The model prompt (in app.py LUNA_INSTRUCTION and character.py) automatically lists current custom emote names, so new ones will be available to Luna without code changes.
- To override a built-in (e.g. make "happy" use your new art), just create a custom with the exact name "happy".
